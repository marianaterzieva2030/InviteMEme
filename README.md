# Presentation Invitation Generator

Уеб приложение за създаване и управление на покани за студентски презентации.

Проектът е разработен по дисциплината "Web технологии" и позволява генериране на стандартни и meme покани за представяне на реферати и проекти.

## Основни функционалности

### Генериране на покани
- Създаване на стандартна покана по шаблон
- Избор между различни визуални теми
- Преглед на поканата в реално време

### Meme Generator
- Избор на meme шаблон
- Добавяне на текст
- Генериране на meme покана

### Управление на покани
- Създаване на нова покана
- Редактиране на съществуваща покана
- Изтриване на покана
- Преглед на всички покани

### Въпроси към презентация
- Добавяне на въпроси
- Преглед на зададени въпроси
- Отбелязване на отговорени въпроси

### Справки и статистики
- Изпратил / не изпратил покана
- Брой създадени покани
- Използвани шаблони

## Инсталация чрез XAMPP

### 1. Клониране на проекта

```bash
git clone https://github.com/marianaterzieva2030/InviteMEme.git
```

### 2. Поставяне на проекта

#### Windows

```text
C:\xampp\htdocs\
```

#### macOS

```text
/Applications/XAMPP/xamppfiles/htdocs/
```

### 3. Стартиране на XAMPP

Стартирайте:

- Apache
- MySQL

### 4. Създаване на база данни

Отворете:

```text
http://localhost/phpmyadmin
```

Създайте база данни:

```sql
invitation_generator
```

Импортирайте:

```text
database/schema.sql
```

### 5. Конфигуриране

Редактирайте:

```php
php/config.php
```

Пример:

```php
$host = "localhost";
$dbname = "invitation_generator";
$user = "root";
$password = "";
```

### 6. Стартиране

Отворете:

```text
http://localhost/InviteMEme
```
