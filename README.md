# InviteMEme

Уеб приложение за създаване и изпращане на покани за студентски презентации.

Проектът е разработен по дисциплината "Web технологии" и позволява генериране на стандартни и meme покани за представяне на реферати и проекти.

## Основни функционалности

### Генериране на покани
- Създаване на стандартна покана по шаблон
- Избор между различни визуални теми
- Преглед на поканата в реално време

### Meme Generator
- Избор на meme шаблон
- Добавяне на текст
- Генериране на meme покана

### Управление на покани
- Създаване на нова покана
- Редактиране на съществуваща покана
- Изтриване на покана
- Преглед на всички покани

### Справки и статистики
- Изпратил / не изпратил покана
- Брой създадени покани
- Използвани шаблони
